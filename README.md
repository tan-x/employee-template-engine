# Employee Template Engine

By [tan-x](https://github.com/tan-x)

## Description

Generates a simple web page of your team with an inquirer user input. Role selection of Manager, Engineer, and Intern with other personal info.

### Tech

- HTML/CSS
- Javascript
- node.js
